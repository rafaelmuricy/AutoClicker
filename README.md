# AutoClicker

I decided to create an auto clicker that clicks in any point of the screen for a simple game I play. I started using WinForms and in the middle of the process I decided to explore different UI frameworks.

In this solution you'll find WinForms, WPF, MAUI, Avalonia, Uno and WinUI3 implementations of the same auto clicker. The code is pretty much the same in all of them, the only difference is the UI framework used.

WinUI does not work, as it runs on sandbox and does not have access to the mouse events.
